📖 عن المشروع | About
تطبيق تعليمي تفاعلي شامل لدرس الاضمحلال الإشعاعي النووي — الفيزياء للصف الثاني عشر.يغطي التطبيق المفاهيم الأساسية من خلال 9 أقسام متكاملة تتضمن:

محاكاة حية لانبعاث الإشعاعات وقدرتها على النفاذ عبر المواد
رسوم متحركة تفاعلية لعمليات الاضمحلال النووي
تمثيلات بيانية تفاعلية لعمر النصف والنشاطية
نمذجة الاضمحلال بالعملات المعدنية
مخططات سلاسل الاضمحلال على رسم N-Z
أسئلة تقييمية تفاعلية مع تغذية راجعة فورية

📚 الأقسام | Sections
🏠 الصفحة الرئيسية — نظرة عامة ونواتج التعلم
👨‍🔬 العلماء والاكتشافات — بيكريل، كوري، رذرفورد، فيلار (تايملاين تفاعلي)
⚛️ أنواع الإشعاعات — مقارنة ألفا/بيتا/غاما مع محاكاة نفاذ تفاعلية
🔄 أنواع الاضمحلال — ألفا، β⁻، β⁺، غاما مع معادلات ورسوم متحركة
🎲 نمذجة الاضمحلال — تجربة العملات المعدنية (عمر النصف)

🔗 سلاسل الاضمحلال — سلاسل U-238 و Th-232 و Ac-235 على مخطط N-Z
🔬 الربط بالتكنولوجيا — 6 تطبيقات حقيقية (إنذار حريق، PET، علاج سرطان...)
📝 مراجعة الدرس — أسئلة MCQ + أكمل المعادلات + مسائل حسابية
